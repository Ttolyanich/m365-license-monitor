Аутентификация Microsoft 365 (Entra ID)

Зарегистрируйте приложение в портале Azure AD / Entra ID и выдайте API разрешения: User.Read.All, Group.Read.All, Directory.Read.All. Для отправки писем через API также требуется Mail.Send.

Интеграция с Jira Cloud

Сбор данных о пользователях Atlassian Cloud. Для работы требуется API Token администратора.

При активации фоновая и ручная синхронизации будут запрашивать данные из Jira.
